# 📧 Spam Email Detection using Machine Learning

A Machine Learning project that classifies emails as **Spam** or **Ham (Not Spam)** using the **Multinomial Naïve Bayes** algorithm. This project was developed as part of my Artificial Intelligence with Python internship and demonstrates the complete workflow of email spam classification using Python and Scikit-learn.

---

## 📖 Project Overview

Spam emails are one of the biggest challenges in digital communication. This project uses Machine Learning techniques to automatically classify emails into:

- 📩 Ham (Legitimate Email)
- 🚫 Spam (Unwanted Email)

The model is trained on a preprocessed email dataset and evaluated using standard machine learning metrics.

## ⚙️ Machine Learning Workflow

1. Import required libraries
2. Load the email dataset
3. Prepare features and labels
4. Split the dataset into training and testing sets
5. Train the Multinomial Naïve Bayes model
6. Predict spam and ham emails
7. Evaluate model performance
